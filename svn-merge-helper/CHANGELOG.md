# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-04-08

### Added

- **進階搜尋**：支援正規表達式 (RegExp) 過濾，並於搜尋框內提供範例提示圖示。
- **搜尋保護**：增加非法正規表達式偵測，當輸入錯誤時會顯示警告並自動退回純文字搜尋。
- **重複合併功能 (Cherry-pick)**：解除已合併 Revision 的勾選限制，允許在特殊情況下手動重新合併。

### Changed

- **選取視覺優化**：被選取的「已合併列」改為橘色背景與左側邊條提示，並恢復亮度，與一般選取狀態 (藍色) 做區分。
- **全選邏輯**：點擊「全選」時，系統會自動跳過已合併的項目，避免誤傳舊紀錄。
- **搜尋說明改善**：將 RegExp 幫助圖示移至搜尋框內部右側，確保佈局在字串變動時保持穩定。

## [1.0.0] - 2026-04-07

### Added

- **專案管理**：支援掃描並匯入 SVN 工作目錄 (Working Copy)。
- **SVN 合併核心**：自動辨識來源與目標路徑，並提供視覺化版本勾選介面。
- **衝突處理**：自動偵測衝突並支援開啟外部工具 (TortoiseMerge) 進行修復。
- **同步更新**：支援單一專案更新及全專案批次更新 (SVN Update)。
- **個人化設定**：內建專案參與批次更新開關，以及自定義外部合併工具路徑。
